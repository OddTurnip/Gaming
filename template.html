Three-column header layout example

Template Overview

This template demonstrates the standard page layout for Gaming Tools pages with theme support.

Header Structure

The header uses a three-column grid layout:

  • Left: Navigation button (← Back) - leave empty for root pages
  • Center: Page title and optional subtitle
  • Right: Theme selector with animated toggle

Required Imports

Add these to your <head>:

<link rel="stylesheet" href="Themes/themes.css">
<script src="Themes/theme-init.js"></script>

Add this before </body>:

<script type="module">
    import { addThemeSelector } from './Themes/theme-manager.js';
    import { createSeasonalEffects } from './Themes/snowflakes.js';

    window.createSeasonalEffects = createSeasonalEffects;
    document.addEventListener('DOMContentLoaded', addThemeSelector);
</script>

addThemeSelector() auto-detects the layout:

  • Three-column: looks for .header-theme or #theme-slot
  • Legacy: creates .header-top-row inside .header

CSS Variables

Use these CSS variables for theme-aware styling:

  • --bg-gradient - Page background gradient
  • --container-bg - Container background color
  • --text-primary - Main text color
  • --text-secondary - Muted text color
  • --accent-primary - Primary accent color
  • --button-gradient-start/end - Button/header gradients
