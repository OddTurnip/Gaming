Template Overview

This template demonstrates the standard page layout for Gaming Tools pages with theme support.

Header Structure

The header uses a three-column CSS grid layout:

  • Left: Navigation button (← Back) - leave empty for root pages
  • Center: Page title and optional subtitle
  • Right: Theme selector with animated toggle

Required Imports

Add these to your <head>:

<link rel="stylesheet" href="Themes/themes.css">
<script src="Themes/theme-init.js"></script>

Add this before </body>:

<script type="module">
    import { autoInitThemeSelector } from './Themes/theme-setup.js';
    autoInitThemeSelector();
</script>

For subdirectories, adjust the path:

<script type="module">
    import { autoInitThemeSelector } from '../Themes/theme-setup.js';
    autoInitThemeSelector();
</script>

Theme Setup Functions

The theme-setup.js module provides two functions:

  • autoInitThemeSelector() - Waits for DOMContentLoaded, then initializes
  • initThemeSelector() - Initializes immediately (use when DOM is ready)

Both functions look for #theme-slot to inject the theme selector.

CSS Variables

Use these CSS variables for theme-aware styling:

  • --bg-gradient-start/end - Page background gradient
  • --container-bg - Container background color
  • --text-primary - Main text color
  • --text-secondary - Muted text color
  • --accent-primary - Primary accent color
  • --button-gradient-start/end - Button/header gradients
  • --bg-white, --bg-light, --bg-hover - Background variations
